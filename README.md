# PRATIK SAHA — Portfolio Website

Ultra-premium personal portfolio for Pratik Saha — Creative Director, Designer & Photographer based in Dhaka, Bangladesh.

## 🚀 Deploy to Vercel

**One-click deploy:**
1. Upload this folder to [vercel.com](https://vercel.com)
2. Drag & drop the zip, or connect via GitHub
3. Vercel auto-detects as static site — click **Deploy**

**CLI deploy:**
```bash
npm i -g vercel
cd pratik-saha-portfolio
vercel --prod
```

## ✨ Features

- **Cinematic preloader** with scan-line animation
- **Animated hero** — particle canvas background, staggered word reveal
- **Ultimate Scroll Text** — scroll-driven word highlight
- **3D Masonry Gallery** — 4 view modes: WALL · TABLE · TUNNEL · FLAT
  - Full lightbox with keyboard navigation
  - 3D tilt on hover
- **3D Curved Carousel** — drag/swipe, auto-play
- **3D Loop Carousel** — auto-rotating testimonials ring
- **Skills tabs** — 4 categories, animated panel switching
- **Recognition logos** — infinite marquee with your 15 logos
- **Contact form** — ready to wire up
- **Lenis smooth scroll** + **GSAP ScrollTrigger** animations
- **Custom cursor** with view/hover states
- **Magnetic buttons**
- **Fully responsive** — mobile, tablet, desktop

## 🎨 Personalize

Replace placeholder content in `index.html`:
- **Projects**: Update `GALLERY_DATA` array (images + descriptions)
- **Carousel**: Update `CC_DATA` array
- **Testimonials**: Update `LC_DATA` array
- **Social links**: Update Instagram URL `@_pratiksahaa_`
- **Images**: Replace files in `assets/` folders with your real work
